Selectable

Install

npm

npm install nestablejs --save

Bower

bower install nestablejs --save

Browser

Grab the file from one of the CDNs and include it in your page:

https://unpkg.com/nestablejs@latest/dist/nestable.js

or

https://cdn.jsdelivr.net/npm/nestablejs@latest/dist/nestable.js

You can replace latest with the required release number if needed.

Initialise

This will create a new instance from the "#myList" element.

<ol id="myList">
    <li class="item">Item 1</li>
    <li class="item">Item 2</li>
    <li class="item">Item 3</li>
    <li class="item">Item 4</li>
    <li class="item">Item 5</li>
    <li class="item">Item 6</li>
</ol>

const instance = new Nestable("#myList");
